Search Configuration

Configure search facets and synonyms to improve search functionality
Search Settings
Define search filters as JSON. Each key represents a facet category with an array of possible values.
Enter related terms on each line, separated by commas. This helps users find items using different terminology.

Example Configuration

{ "categories": ["Electronics", "Books", "Clothing"], "brands": ["Apple", "Samsung", "Sony"], "price_range": ["0-50", "50-100", "100+"] } // Synonyms example: phone,mobile,smartphone,cellphone laptop,computer,notebook,pc car,vehicle,automobile,auto
